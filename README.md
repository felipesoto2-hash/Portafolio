# Predicción del nivel de PIB utilizando datos del Banco Mundial

Este proyecto tiene como objetivo analizar y predecir el nivel del Producto Interno Bruto (PIB) de distintos países a partir de indicadores económicos, sociales y demográficos obtenidos desde la base World Development Indicators (WDI) del Banco Mundial. La información es extraída directamente mediante la API oficial del Banco Mundial, lo que permite construir una base de datos reproducible y actualizada. A partir de un análisis descriptivo inicial, se revisa la estructura del dataset, se tratan los datos faltantes y se discretiza el PIB en categorías ordenadas. Posteriormente, se aplica Análisis de Componentes Principales (PCA) para reducir la dimensionalidad de las variables explicativas, y finalmente se utilizan modelos de clasificación para predecir el nivel de PIB de los países.
